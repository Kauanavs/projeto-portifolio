# 💻 Portfólio Desenvolvedor Front-end

Bem-vindo ao repositório do meu portfólio pessoal. Este projeto foi desenvolvido para demonstrar minhas habilidades em desenvolvimento front-end moderno, focando em performance, acessibilidade e design de alta conversão.

## 🎨 Visualização

O design foi inspirado em agências digitais e startups de tecnologia modernas, tipografia moderna (`Outfit` e `Inter`), e microinterações de scroll e hover.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído **sem frameworks pesados**, utilizando a base da web para garantir máxima performance:

*   **HTML5:** Estrutura semântica e limpa.
*   **CSS3:** Variáveis (Custom Properties), CSS Grid, Flexbox, Animações e Design Responsivo (Mobile First).
*   **JavaScript (Vanilla):** Interatividade do menu, scroll spy e IntersectionObserver para animações.
*   **Phosphor Icons:** Biblioteca de ícones moderna.
*   **Google Fonts:** Para tipografia elegante.

## 📁 Estrutura de Arquivos

## ✨ Funcionalidades

- **Design Premium:** Interface moderna, escura e com alto contraste.
- **Totalmente Responsivo:** Adaptado para Mobile, Tablets e Desktops.
- **Animações "Scroll Reveal":** Elementos aparecem suavemente à medida que você rola a página.
- **Header Dinâmico:** Barra de navegação com efeito "Glassmorphism" que muda ao dar scroll.
- **Formulário de Contato:** Simulação de envio com feedback visual (Botão com status).
- **Código Limpo:** Facilmente personalizável com uso de variáveis CSS.
