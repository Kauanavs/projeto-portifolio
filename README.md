# 💼 Kauan Lucas — Portfólio Front-end

Portfólio pessoal desenvolvido para apresentar minhas habilidades, projetos reais e facilitar o contato com recrutadores e empresas.

🔗 **[Ver portfólio online](https://kauanavs.github.io/projeto-portifolio/)**

---

## 👨‍💻 Sobre o Projeto

Este portfólio foi construído do zero com HTML5, CSS3 e JavaScript vanilla — sem frameworks, sem templates. Cada detalhe foi pensado para transmitir profissionalismo, clareza e foco em front-end.

**Design inspirado em:** portfólios de agências digitais e desenvolvedores sêniors do mercado.

---

## 🚀 Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| **HTML5** | Estrutura semântica e acessível |
| **CSS3** | Variables, Grid, Flexbox, animações, responsividade |
| **JavaScript (Vanilla)** | IntersectionObserver, menu mobile, WhatsApp redirect |
| **Google Fonts** | Syne (display) + DM Sans (corpo) |

---

## ✨ Funcionalidades

- ✅ **Hero Section** com badge de disponibilidade, stats e botões de ação
- ✅ **Seção Sobre** com card de perfil, tech stack e destaques profissionais
- ✅ **Habilidades** com cards animados e barras de progresso
- ✅ **4 Projetos reais** com links Live Demo + GitHub
- ✅ **Contato** com botão direto para WhatsApp e opção de e-mail
- ✅ **GitHub + LinkedIn** visíveis no nav, hero e footer
- ✅ **Animações scroll** suaves via IntersectionObserver
- ✅ **Header glassmorphism** que muda ao rolar
- ✅ **Menu mobile** funcional
- ✅ **Back to top** button
- ✅ **Totalmente responsivo** (mobile, tablet, desktop)

## ⚙️ Como Personalizar

1. Clone o repositório
2. No `index.html`, atualize os links e textos com suas informações
3. No `js/script.js`, substitua `WHATSAPP_NUMBER` pelo seu número real (formato: `55DDD9XXXXXXXX`)
4. Substitua `seuemail@gmail.com` pelo seu e-mail real na seção de contato
5. Abra `index.html` no navegador (recomendado: Live Server no VSCode)

---

## 🌐 Deploy

### GitHub Pages
1. Suba o código para um repositório no GitHub
2. Vá em **Settings → Pages**
3. Selecione branch `main` e clique em Save
4. O link será gerado automaticamente

### Vercel (alternativa)
1. Importe o repositório na [Vercel](https://vercel.com/)
2. Clique em Deploy — pronto!
